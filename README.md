# First To 100

A real-time multiplayer game built as a side project to explore monorepos,
lobby systems, and live gameplay mechanics.

## What's inside?

This Turborepo includes the following packages/apps:

### Apps and Packages

- `server`: a [Fastify](https://fastify.dev/) app
- `web`: a [Next.js](https://nextjs.org/) app
- `@repo/ui`: a stub React component library for `web` applications
- `@repo/typescript-config`: `tsconfig.json`s used throughout the monorepo

Each package/app is 100% [TypeScript](https://www.typescriptlang.org/).
